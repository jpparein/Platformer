# 🎮 Platformer Unity – Série YouTube

Projet Unity 2D utilisé comme support d’une série de tutoriels publiée régulièrement.

![Aperçu du projet](images/preview.png)

---

## 📺 Suivi de la série

Ce projet évolue au fil des vidéos.

👉 Chaque mise à jour correspond à une nouvelle étape :
- ajout de fonctionnalités
- amélioration du code
- évolution de la structure du projet

---

## 🚀 État actuel

Le projet contient la version la plus récente correspondant à la dernière vidéo publiée.

---

## 📦 Récupération du projet

1. Télécharger ou cloner ce dépôt
2. Ouvrir avec Unity Hub
3. Lancer la scène dans `Assets/Scenes`

---

## 📁 Organisation (évolutive)

La structure peut évoluer selon les besoins du projet :

- `Assets/Scripts`
- `Assets/Scenes`
- `Assets/Prefabs`
- `Assets/Art`
- `Assets/Audio`
